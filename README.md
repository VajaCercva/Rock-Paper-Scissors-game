# Rock-Paper-Scissors-game
ეს არის ჩვეულებრივი ჯეირანის თამაში სადაც თამაში იქამდე მიმდინარეობს სანამ მოთამაშე ან მოწინააღმდეგე(ai) არ დააგროვებს 3 ქულას
